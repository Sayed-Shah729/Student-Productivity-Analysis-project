# 📊 Student Productivity Analysis & Performance Prediction

## 📌 Project Overview

This project focuses on analyzing student productivity and predicting academic performance using a dataset of 5000 students. The analysis was performed using Python in Google Colab.

---

## ⚙️ Tools & Technologies

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📁 Dataset

The dataset contains 5000 student records with 21 features including:

* Study hours
* Social media usage
* Sleep hours
* Exercise
* Productivity score
* Exam score

---

## 🔍 Analysis Performed

* Data loading and cleaning
* Exploratory Data Analysis (EDA)
* Data visualization
* Relationship analysis between variables

---

## 📈 Model

* Linear Regression used for prediction
* Train-test split applied

---

## 📊 Evaluation Metrics

* R² Score
* Mean Absolute Error (MAE)

---

## 💡 Key Insights

* Study hours positively impact performance
* High social media usage reduces productivity
* Balanced lifestyle improves academic results
